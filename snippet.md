# outln

CLI for generating structured code outlines. Quick codebase understanding for code review, API discovery, and documentation.

## Usage

```bash
npm install -g outln
outln <file-path>
```

## Supported

TypeScript, JavaScript, Java, Kotlin, C#, Go, Rust, Markdown

## Use Cases

- Codebase structure exploration
- Finding classes/functions
- Code review preparation
- API endpoint discovery
